SIMULATION FORM

CROP FORM

This form specifies the resources available once the simulation begins. It is mandatory to fill this form before beginning the simulation. On the drop down fields (1.), you must first select the option you want to use/buy and then the field on the right (2.) needs to be used to specify the number of vehicles, workers etc. The button on the bottom (3.) resets all the fields on the first form in case it is needed and the button on the side of that (4.) submits the form

This form is used to create and delete crops on the map in the right, to begin the simulation, at least one crop field must be created. The first step to create a new crop is clicking the desired area on the map (1.), then the position fields (3.) will be automatically filed ad you will have to specify the size of the crop in the field (2.). Finally, the accept (4.) or cancel (5.) buttons should be pressed to create, cancel or delete the crop in case you have selected an already created crop.

SIMULATION FORM

CROP FORM

After filling the simulation setup data and creating at leas one crop field, you can press the play button (1.) on the top right corner to begin the simulation process and go to the main simulation screen.

This screen shows the status of the simulation on real time giving data about the balance and the state of the crop. The buttons on the top right corner can be used to pause/play (1.) or accelerate (2.) the simulation depending on the need. On the left side menu, the balance (3.) expenses and income can be viewed and under that, the status of each crop is displayed in real time (4.). Finally the arrow on the side (5.) can be pressed to edit the simulation data.
